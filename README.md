## ğŸ§ª Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Preveen369/preveen_portfolio_app.git
   cd preveen_portfolio_app
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

   The application will be available at `http://localhost:5173` (or the port shown in your terminal).

4. **Build for production**

   ```bash
   npm run build
   ```

5. **Deploy to Netlify**

   - **Option 1**: Connect your GitHub repository to Netlify for automatic deployments
     - Go to [Netlify](https://netlify.com)
     - Click "New site from Git"
     - Connect your GitHub account and select this repository
     - Set build command: `npm run build`
     - Set publish directory: `dist`

   - **Option 2**: Deploy using Netlify CLI

     ```bash
     npm install -g netlify-cli
     netlify deploy --prod
     ```

   ğŸŒ Access your app live at: [Preveen Portfolio App](https://preveens-portfolio-app.netlify.app/)
